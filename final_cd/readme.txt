SNMP/XML gate
--------------

Struktura adresářů:
/source - zdrojové soubory brány
/manager - zdrojové soubory manažerské aplikace
/config_file - ukázkový konfigurační soubor brány
/init_script - spouštěcí script pro démona brány

Text:
/text/source - zdrojové soubory textu ve formátu LaTeX
/text/pdf - text ve formátu pdf

Soubory:
gate_usrguide.txt - instalační a uživatelská příručka protokolové brány
manager_usrguide.txt - instalční a uživatelská příručka manažerské aplikace
